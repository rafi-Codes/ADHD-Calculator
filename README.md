Users answer a series of questions like:

> *"How often do you feel restless or fidget with your hands/feet when seated?"*

Each answer maps to a score (Never = 0, Very Often = 4). The total score is used to display a feedback message such as:

- **Low score:** Few ADHD-like traits observed  
- **Moderate score:** Some traits present; consider learning more  
- **High score:** Many traits observed; consider consulting a professional
